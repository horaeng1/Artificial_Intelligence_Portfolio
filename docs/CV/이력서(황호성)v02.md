## 프로젝트

### Nowyouseeme – 한국어 자동 화면 해설(AD) 웹 서비스 (팀, 5주)

- **역할**: 팀 PM, 백엔드/프론트엔드 핵심 구현, LLM·멀티모달 파이프라인 설계
- **개요**: 시각장애인을 위해 영상을 입력하면 **대사와 겹치지 않는 무음 구간에 한국어 AD를 자동 생성·삽입**하는 웹 서비스 MVP 구현
- **주요 작업**
  - Silero VAD + Whisper + SigLip으로 **대사·무음 구간 및 의미 단위 청크** 탐지
  - 최근 대사, 이전 AD, 키프레임 설명, 무음 길이를 묶는 **컨텍스트 윈도우** 설계
  - Gemini / GPT / Cookie(Qwen-VL+Llama) / Jack(Gemini 멀티스테이지)을  
    `input: 청크 JSON → output: AD JSON` 규격으로 통합해 모델 교체 가능한 구조 구현
  - AD 도메인 가이드를 반영한 **프롬프트 템플릿** 설계: 현재 시제, 객관 서술, 대사 중복 금지, 길이/시간 제약
  - gTTS/Coqui XTTS + FFmpeg로 AD 음성을 합성하고, 무음 구간에만 삽입되는 타임라인 후처리 구현
  - Notion Functional Requirements(FR-01~FR-31) 설계로 업로드·ASR/VAD·AD 생성·TTS까지 **엔드투엔드 기능 우선순위·상태 관리**
- **성과**
  - 약 **30편**의 영상을 대상으로 테스트, 5분 영상 기준 처리 시간:  
    - Gemini 약 **1분**, GPT 약 **5분**, Cookie 약 **20분**  
  - 동일 입력에 대해 모델별 품질·속도·비용 트레이드오프를 비교 가능한 평가 환경 구축
  - 대사 구간 보호 + 무음 기반 AD 삽입으로, AD가 원본 대사를 덮지 않는 타임라인 확보
- **Repo / Demo**
  - GitHub: https://github.com/horaeng1/Nowyouseeme (구조·파이프라인·배포 README 정리)
